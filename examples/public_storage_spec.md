# Public Storage Account (Bluerunner InsightCloudSec Findings)

## Description

As an InfoSec stakeholder,
I want Bluerunner storage accounts to be secured and not publicly accessible,
So that we comply with security standards and prevent unauthorized access.

As a system admin,
I want to disable public access on all identified Bluerunner storage accounts,
So that data is only accessible through secure, authenticated channels.

## Acceptance Criteria

- Public access is disabled for all accounts.
- Verification that applications using these accounts still function correctly.
