# Shopping Cart Discount Feature

As a RegisteredUser, I want to apply a coupon code to my Cart, so that I can see the discounted total.

## Guidelines
1. The RegisteredUser must enter a valid coupon code in the voucher field.
2. If the coupon is active, the system should apply the discount percentage immediately.
3. Only one coupon can be applied per checkout transaction.
4. The system must recalculate the total cart price and display the discounted price fast.

## Checkout
The user is redirected to the checkout page after coupon application.
